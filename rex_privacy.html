Política de Privacidade – REx: Research Explorer

Esta Política de Privacidade descreve como a extensão REx – Research Explorer Extension ("nós" ou "a extensão") trata as informações dos usuários.

1. Coleta de Informações

O REx NÃO coleta, armazena ou transmite quaisquer dados pessoais identificáveis.

  • A extensão analisa o texto das páginas que você visita localmente no seu navegador apenas para identificar padrões de ISSN (ex: 1234-5678).
  • Remote Data Fetching: A extensão baixa a base de dados Qualis publicamente disponível no domínio pconradjunior.github.io. Esta conexão é usada exclusivamente para baixar o arquivo de dados e não envia nenhuma informação pessoal ou histórico de navegação para o servidor.
  • Nenhuma URL visitada ou conteúdo de página é enviado para qualquer servidor externo.

2. Uso de Dados

  • Processamento Local: Todas as consultas e verificações de dados Qualis são realizadas via IndexedDB, um banco de dados local armazenado permanentemente no seu dispositivo.
  • Sem Rastreamento: Não utilizamos cookies de rastreamento, Google Analytics ou qualquer outra ferramenta de análise de terceiros.

3. Permissões Requeridas

  • activeTab / scripting: Necessárias para identificar ISSNs nas páginas que você decide visualizar.
  • storage: Utilizada apenas para gerenciar o banco de dados Qualis local.
  • host_permissions (<all_urls>): Necessária para que a detecção funcione em qualquer site acadêmico ou de busca técnica.

4. Compartilhamento de Dados

Como não coletamos dados, não há compartilhamento de informações com terceiros em nenhuma circunstância.

5. Alterações

Esta política pode ser atualizada ocasionalmente, sendo que a versão mais recente estará sempre disponível no repositório do projeto.

6. Contato

Se você tiver dúvidas sobre esta política, sinta-se à vontade para entrar em contato.
